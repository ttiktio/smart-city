# 🏙️ Nexus City - Smart City Simulator 3D

**Nexus City** คือเกมจำลองการบริหารจัดการเมืองอัจฉริยะแบบ 3 มิติ (3D Smart City Simulator) ที่พัฒนาบนเว็บบราวเซอร์โดยใช้ **React, TypeScript, Three.js (React Three Fiber)** และมีแผงสถิติเมืองแบบกระจกโปร่งแสง (Glassmorphism Dashboard) ที่ออกแบบมาอย่างสวยงามล้ำสมัย

ผู้เล่นสามารถสวมบทบาทเป็นนายกเทศมนตรีเพื่ออัปเกรดตึกอาคาร ติดตั้งพลังงานสะอาด (Solar Panel / Wind Turbine) ดูแลคุณภาพชีวิตประชากร จัดการปัญหาจราจร และรับมือกับภัยพิบัติต่างๆ แบบ Real-time

---

## 🛠️ เทคโนโลยีที่ใช้งาน (Tech Stack)
* **Frontend**: React 19 + TypeScript + Vite
* **3D Engine**: Three.js, `@react-three/fiber`, `@react-three/drei`
* **Post-Processing (แสงและเอฟเฟกต์)**: `@react-three/postprocessing` (ใช้ Bloom และ Vignette เพื่อเพิ่มความสมจริง)
* **State & Simulation**: `zustand` (ควบคุมลูปเวลา, สภาพอากาศ, และความต้องการของประชากร)
* **Charts**: `recharts` (วาดกราฟสถิติเมืองในแบบ Real-time)
* **Styling**: Tailwind CSS + Glassmorphism UI

## ⚙️ ขั้นตอนการเริ่มใช้งานและการติดตั้ง (Setup & Running)

เพื่อให้มั่นใจว่าสภาพแวดล้อมพร้อมใช้งาน ให้ทำตามขั้นตอนดังต่อไปนี้:

### 1. ติดตั้ง Dependencies
เปิด Terminal ในโฟลเดอร์โปรเจกต์นี้และรันคำสั่ง:
```bash
npm install
```

### 2. การรันในโหมดพัฒนา (Development Mode)
หากต้องการรันเพื่อเข้าเล่น ตรวจดูโค้ด หรือทำการแก้ไขฟีเจอร์เพิ่มเติม:
```bash
npm run dev
```
หลังจากรันคำสั่ง ระบบจะแสดงลิงก์โลคอลเซิร์ฟเวอร์ ให้เข้าใช้งานผ่านบราวเซอร์ (ปกติคือ `http://localhost:5173`)

## 🔍 การตรวจสอบและทดสอบ Build (Build Verification)

เพื่อรับรองว่าโค้ดไม่มีความผิดพลาดทางโครงสร้าง Type และพร้อมสำหรับการ Deploy ขึ้น Production ให้รันขั้นตอนการ Verify ดังต่อไปนี้:

### 1. ตรวจสอบข้อผิดพลาดทางด้าน Type (Typecheck)
รันคำสั่งนี้เพื่อสแกนหา Error ของ TypeScript ทั้งหมดในโปรเจกต์:
```bash
npx tsc --noEmit
```
*(ถ้าไม่มีข้อความ Error แสดงขึ้นมา แสดงว่า Type ของซอร์สโค้ดถูกต้องสมบูรณ์ 100%)*

### 2. การ Build สำหรับใช้งานจริง (Production Build)
ทำหน้าที่คอมไพล์โค้ด มินิฟายไฟล์ (minify) และย่อยเป็นไฟล์สำหรับขึ้นเซิร์ฟเวอร์จริงลงในโฟลเดอร์ `dist`:
```bash
npm run build
```
เมื่อสร้างเสร็จ ระบบจะแสดงรายการไฟล์ที่ถูกบีบอัดสำเร็จ เช่น `dist/index.html` และโฟลเดอร์ `dist/assets/`

### 3. รันเพื่อทดสอบไฟล์ตัวอย่างจากการ Build (Build Preview)
หลังจากทำการ Build สำเร็จแล้ว ให้ใช้คำสั่งนี้เพื่อจำลองเว็บบอร์ดเซิร์ฟเวอร์จริงและเปิดดูผลลัพธ์:
```bash
npm run preview
```
จากนั้นเข้าลิงก์ที่แสดงบนหน้าจอ (ปกติคือ `http://localhost:4173`) เพื่อเล่นและเช็กว่าโมเดล 3D รวมถึงหน้าจอ UI โหลดขึ้นมาทำงานได้อย่างสมบูรณ์

## ⌨️ ปุ่มคีย์บอร์ดลัดภายในเกม (In-Game Controls)

ขณะอยู่ในหน้าจอจำลองเมือง คุณสามารถกดปุ่มบนคีย์บอร์ดเพื่อสั่งงานได้ทันที:

| ปุ่มคีย์บอร์ด | คำสั่งทำงาน |
| :--- | :--- |
| `Spacebar` | **หยุด (Pause) / เล่นต่อ (Resume)** การจำลองเวลาในเมือง |
| `1` | สลับเป็นมุมกล้อง **หมุนอิสระ (Free Orbit)** (ใช้เมาส์หมุนและซูมได้ตามใจชอบ) |
| `2` | สลับเป็นมุมกล้อง **มุมมองจากด้านบน (Top-Down)** |
| `3` | สลับเป็นมุมกล้อง **ติดตามรถยนต์ (Follow Vehicle)** ที่วิ่งอยู่บนถนน |
| `D` | เปิด / ปิด แผง **Dashboard กราฟสถิติ** ด้านซ้าย |
| `M` | เปิด / ปิด แผง **ภารกิจ (Missions)** ด้านขวา |
| `A` | เปิด / ปิด **เสียงบรรยากาศของเมือง** (ลม, ฝน, และเสียงเมือง) |
| `Escape` | ยกเลิกการเลือกอาคารที่กำลังไฮไลต์อยู่ |
| `คลิกซ้ายอาคาร` | เปิดรายละเอียดอาคารนั้นเพื่อทำการอัปเกรด ปิด/เปิดไฟฟ้า หรือติดตั้งแผงโซลาร์ |
